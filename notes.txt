GitHub
git init - создание нового репозитория
git add . - добавить все изменения в индекс
git commit - после индекса сохранить в репо
git checkout <название ветки> - переключиться на нужную ветку
git checkout -b <branch name> - создание новой втеки и переход в нее
git branch - отображает список всех веток
git branch -m <new branch name> - переименовать текущую ветку
git branch -d <branch name> - удалить ветку
git commit -m "<message>" - создание коммита с записью изменений в репо.
git push - запушить на github
git pull - скачать с github
git clone <url> - клонирование удаленного репозитория в локальный
git push -u origin <branch> - загрузка изменений из локальной ветки в удаленную с созданием связи между ними.
